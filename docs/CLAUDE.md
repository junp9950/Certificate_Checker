## 기술적 개선사항 (v5.1)
- `tkinterdnd2` 라이브러리를 통한 실제 드래그 앤 드롭 구현
- DN 정규화 알고리즘으로 인증서 체인 연결성 향상
- 5단계 연결성 검증: 정확매칭 → 정규화매칭 → CN매칭 → 부분매칭 → 연결끊김
- 파일 유효성 검증: 매직바이트 + 크기 + 확장자 통합 검증

- 크로스 플랫폼 호환성 (Windows/Linux/macOS)
